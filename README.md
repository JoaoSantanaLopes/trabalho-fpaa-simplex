# Implementação do Algoritmo Simplex

Implementação em Java do algoritmo Simplex, seguindo a formulação apresentada no
livro CLRS (Introduction to Algorithms), para a disciplina de Fundamentos de
Projeto e Análise de Algoritmos.

## Referências

- *CLRS — Introduction to Algorithms (3ª edição)*, Cormen, Leiserson, Rivest e
  Stein. Capítulo 29 (Linear Programming). Referência principal para a formulação
  do algoritmo (forma de folga e os procedimentos PIVOT, SIMPLEX e
  INITIALIZE-SIMPLEX).

- *Repositório lvlingxiao1/SimplexLinearProgramSolver* — implementação em
  Python que segue o CLRS, usada como base de lógica e da decisão de manter a
  matriz A com sinal natural. Disponível em:
  https://github.com/lvlingxiao1/SimplexLinearProgramSolver

- *Soluções dos exercícios do Capítulo 29 (walkccc)* — apoio na compreensão dos
  procedimentos. Disponível em: https://walkccc.me/CLRS/Chap29/

A implementação em Java, a organização em classes e os comentários são de autoria
própria, a partir das referências acima.
